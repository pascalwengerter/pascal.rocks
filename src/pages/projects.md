## Current

### Open Mat Radar
Trying to help other Jiu-Jitsu practicioners find open mats nearby and connect with the community. Built using [Nuxt](https://www.nuxt.com) and [Supabase](https://www.supabase.com).
You can find it running here: [Open Mat Radar](https://open-mat-radar.com).

---

### MuscleMate.app
Frustrated with tracking- and advertisement-heavy workout apps, I have decided to create my own, very basic solution. It's built with [Vue3](https://vuejs.org/) and [TailwindCSS](https://tailwindcss.com/) and deployed on [Vercel](https://vercel.com). You can find it running here: [MuscleMate.app](https://musclemate.app).

---

### Online Tabata Workout Track Creator
Create HIIT soundtracks from your own MP3s. 
You can find it running here: [tabata.MuscleMate.app](https://tabata.musclemate.app).
